================================================================================

QUERY CONSTRUCTION TIPS
================================================================================

1. USE SPECIFIC URLS:
   • Include main URL: https://example.com
   • Better for targeted extraction

2. USE ACTION KEYWORDS:
   • "summarize", "extract", "list", "find", "crawl", "map", "render"
   • These trigger the correct tool combinations

3. BE CLEAR ABOUT INTENT:
   • "summarize" → Extract + Search
   • "list all urls" → Map + Extract
   • "find information about" → Search
   • "compare" → Search + Extract

4. COMBINE KEYWORDS FOR ROUTING:
   • URL + "summarize" → Extract → Search
   • URL + "list urls" → Map → Extract → Search
   • "find" + topic → Search → Search
   • URL + "crawl" → Crawl → Extract → Search

5. EXAMPLES OF WELL-FORMED QUERIES:
   • https://example.com summarize                         [simple]
   • find information about artificial intelligence         [topic search]
   • https://docs.python.org list all pages and extract     [mapping]
   • compare machine learning frameworks                    [comparison]
   • https://github.com/tensorflow/tensorflow explain features [extraction + qa]

================================================================================
TOOL REFERENCE
================================================================================

SEARCH:     Web search with AI-powered answer generation
            Best for: Broad topic research, trending information
            
CRAWL:      Traverse website structure and extract from multiple pages
            Best for: Multi-page analysis, site structure discovery
            
EXTRACT:    Pull content from specific URLs
            Best for: Single or multiple URL content extraction
            
MAP:        Discover all URLs on a website
            Best for: Site mapping, link discovery, URL inventory
